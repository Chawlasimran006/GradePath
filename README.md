# GradePath

GradePath is a project I started after noticing that students repeatedly ask the same questions every semester:

* How many marks do I need in the final exam to get an A?
* Can I still achieve my target CGPA?
* What SGPA do I need in the upcoming semesters?

Most of us either calculate these manually or use multiple calculators. GradePath aims to bring these calculations together in one place and make academic planning easier.

## Current Goals

The first version of GradePath will include:

* Internal Marks Predictor
* Final Exam Marks Requirement Calculator
* CGPA Goal Planner
* SGPA Requirement Calculator

## Why I Am Building This

As a B.Tech student, I wanted to build something that solves a real problem students face rather than creating another generic college project.

GradePath focuses on helping students make informed academic decisions using simple calculations and clear insights.
